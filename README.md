Website sederhana menampilkan data steam menggunakan api

## Getting Started

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

buka [http://localhost:3000](http://localhost:3000) dibrowser

kamu bisa mulia mengedit `app/page.js` halaman website akan otomatis berubah
